# Great Learning PGP-AIML Program Projects
This repository contains projects completed as part of the Great Learning Post-Graduate Program in Artificial Intelligence and Machine Learning (PGP AIML). Below is a brief overview of each project:

## Applied Statistics
### Project Description
This project leverages statistical analysis and visualization techniques to address industry problems, analyze past tournament data for investment decisions, and evaluate startups from the Startup Battlefield competition. It is divided into three parts:

- **Answering Industry Problems:** Applying statistical inferences to real-world industry data to make data-driven decisions.
- **Analyzing Tournament Information:** Assessing historical tournament data to identify trends and inform investment decisions.
- **Evaluating Startups:** Analyzing the performance of startups from the Startup Battlefield competition to identify success factors and investment opportunities.

### Skills & Tools Covered
- Exploratory Data Analysis (EDA)
- Data Visualization
- Statistical Inference
- Hypothesis Testing
- Python

## Computer Vision - Face Recognition
### Project Description
This project, part of the Computer Vision course, focuses on object detection and recognition through three subprojects:

- **Human Face Detection Model:** Develop an object detection model to highlight human faces, automating the process of providing cast and crew information while streaming.
- **Training Dataset Curation:** Curate a dataset specifically for training the human face detection model.
- **Face Identification Model:** Implement a face identification model for a company to recognize human faces from images.

### Skills & Tools Covered

## Ensemble Techniques
### Project Description
This project, part of the Ensemble Techniques course, addresses a customer churn issue faced by a telecommunication company. The goal is to understand data patterns and predict which customers are likely to churn based on multiple variables, aiding the company in retaining existing customers. The project involves building an autonomous machine learning workflow that processes a CSV file and identifies the best-performing model.

## Featurization, Model Selection & Tuning
### Project Description
This project, part of the Featurization, Model Selection & Tuning course, involves building and training a prediction model to identify the Pass/Fail yield of a process entity in semiconductor manufacturing. By employing supervised learning, ensemble modeling, and unsupervised learning techniques, the project aims to determine key factors contributing to yield excursions, thereby increasing process throughput, reducing time to learn, and lowering per-unit production costs.

### Skills & Tools Covered
- Supervised Learning
- Principal Component Analysis (PCA)
- Feature Engineering
- Model Tuning
- Grid Search
- Python

## NLP Sequential NLP
### Project Description
This project, part of the Natural Language Processing course, delivers two sub-projects focused on sequential NLP:

- **Text Sentiment Classification for Digital Content & Entertainment:** Implement a sequential NLP-based text classifier to determine customer sentiments based on reviews from the IMDB database. This classifier helps a company in the digital content and entertainment industry to analyze customer feedback.

## Supervised Learning
### Project Description
This project, part of the Supervised Learning course, utilizes popular classification techniques to predict outcomes after extensive Exploratory Data Analysis (EDA) and handling of missing values and data imbalances. The project is divided into two parts:

## Unsupervised Learning
### Project Description
This project, part of the Unsupervised Learning course, involves two parts focusing on different tasks:

- **Car Segmentation:** Segment cars into various categories based on attributes such as fuel consumption and other features. This task involves clustering cars into distinct groups without labeled data.
- **Vehicle Classification:** Classify a given silhouette as one of three types of vehicles using a set of features extracted from the silhouette. The vehicles may be viewed from multiple angles, making this a challenging classification problem.
